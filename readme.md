### 操作建议
1、通过http克隆的方式，clone代码到本地;  
2、实习生在拉去master分支代码后，创建个人分支，分支命名格式：`dev-名字拼音`，例如：`dev-guojincai` 之后所有的编辑更新操作，都在个人分支上处理，不做任何分支的合并。  
3、在个人分支下，可以有序地建立自己的练习文件夹，例如：  

```
|-- project_01
  |--test_01
    |--index.html
    |--images
    |--css
      --index.js
    |--js
     --index.js
  |--test_02
  |--test_03_git
  |--test_04_xxx
  |--......
```
